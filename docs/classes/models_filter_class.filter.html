Class Filter

Filter class The "base" class defining "Filter"

"filter" can be defined in 2 forms:

  • the first is it's the JSON form ( {name:rules}} )
  • the second is its instance form (in the FilterManger})

A filter is first being created in the first form and then gets instantiate to its second form Only then you can use isValid
